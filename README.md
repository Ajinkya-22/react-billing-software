# React Billing Software

This is a simple billing software built using React.js.  
It allows users to add products, enter price and quantity, and automatically calculate the total bill.

## Features
- Add product name
- Enter price and quantity
- Automatic total calculation
- Dynamic bill table
- Clean and simple UI

## Technologies Used
- React.js
- JavaScript
- HTML
- CSS
